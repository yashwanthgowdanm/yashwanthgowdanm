### PhishLens — Neural Phishing and Malicious URL Detection

**PyTorch · 1D CNN · REST API · JavaScript · Adversarial ML · Cybersecurity**

Dual-engine phishing and malicious URL detection system built for ASU CSE 543: Information Assurance and Security. Combines a character-level neural model for generalization with a heuristic fallback engine for known-threat blocking.

**Highlights**

- Designed a character-level 1D CNN for malicious URL classification
- Used raw URL sequences with character embeddings and convolutional feature extraction
- Built a Python REST API and JavaScript operator console
- Stress-tested against homoglyph substitutions, subdomain injections, and delimiter manipulation
- Achieved 92.98% binary accuracy and 0.9636 ROC-AUC
