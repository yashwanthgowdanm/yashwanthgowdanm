### 🚀 ProphetLSTM-GAN — Rocket Engine Health Monitoring
> PyTorch · LSTM · GAN · Anomaly Detection · Time-Series

Hybrid LSTM-GAN deep learning architecture for unsupervised anomaly detection in liquid rocket engine telemetry. Engineered a combined scoring metric that improved fault detection **AUC by 3.7%** over baseline models.

[![Repo](https://img.shields.io/badge/GitHub-181717?style=flat&logo=github&logoColor=white)](https://github.com/yashwanthgowdanm/ProphetLSTM-GAN)

### 🔊 Autonomous Siren Detector
> DSP · STFT · Signal Processing · Real-Time · Python

Deterministic DSP system for emergency vehicle siren detection using Short-Time Fourier Transform (STFT) spectral analysis. Achieves **<20ms latency** at **300m+ range** — fully offline, no ML model required.

[![Repo](https://img.shields.io/badge/GitHub-181717?style=flat&logo=github&logoColor=white)](https://github.com/yashwanthgowdanm/sirendetector-sim)

### ⚙️ PAPI Controller — Solar Irrigation System
> MATLAB · Simulink · Control Theory · Inverse Dynamics · 2-DOF

Designed a 2-DOF hybrid control system using physics-based inverse dynamics for solar-powered pump regulation. Achieved **60% faster rise time** and near-instant disturbance rejection compared to standard PID controllers.

[![Repo](https://img.shields.io/badge/GitHub-181717?style=flat&logo=github&logoColor=white)](https://github.com/yashwanthgowdanm/PAPIController)

---

### 🧠 Hierarchical MPC — Off-Grid Solar Irrigation
> MPC · Python · Adaptive Control · Penman-Monteith · Optimization

Dual-loop MPC architecture combining outer-loop Model Predictive Control with Penman-Monteith evapotranspiration modelling and an inner-loop adaptive PI controller. Outperforms single-loop PID across all simulation scenarios.

[![Repo](https://img.shields.io/badge/GitHub-181717?style=flat&logo=github&logoColor=white)](https://github.com/yashwanthgowdanm/hirearchical-mpc)

### 🐝 Debris Detection Using Swarm Robotics
